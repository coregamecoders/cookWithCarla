# 🍳 cookWithCarla <VIDEO LINK>

Welcome to **cookWithCarla**, a modern and minimal recipe-inspired web project designed to make cooking ideas simple, accessible, and visually enjoyable. This project is part of my personal portfolio as a creative and functional web development experiment focused on building clean user experiences around everyday cooking.

cookWithCarla is inspired by the idea that cooking should not feel complicated or overwhelming. Instead, it should feel intuitive, guided, and even fun. This project aims to bring that idea to life through a lightweight, responsive, and easy-to-navigate web interface.

---

## 🌟 Project Vision

The main vision behind **cookWithCarla** is to create a digital space where users can quickly explore recipes without unnecessary distractions. Many recipe websites today are overloaded with ads, popups, and cluttered layouts. This project takes the opposite approach: simplicity first.

The goal is to:
- Reduce complexity in browsing recipes
- Provide a calm and clean interface
- Focus on usability and clarity
- Make cooking inspiration accessible to everyone

It is designed as both a learning project and a portfolio piece that demonstrates my ability to build structured, responsive, and user-friendly web applications.

---

## 👩‍🍳 Features

cookWithCarla includes several core features that simulate a real-world recipe browsing experience:

### 🍲 Recipe Display System
Recipes are presented in a structured format with clear titles, ingredients, and step-by-step instructions. Each recipe is designed to be easy to read and follow, especially on mobile devices.

### 🔍 Simple Navigation
Users can browse through different recipes using a clean navigation system. The interface avoids unnecessary complexity, focusing only on what is essential.

### 📱 Responsive Design
The layout is fully responsive and adapts to different screen sizes, including smartphones, tablets, and desktop computers. This ensures a consistent experience across all devices.

### 🎨 Minimal UI Design
The visual design is intentionally minimal. It uses soft spacing, readable typography, and simple layout structures to create a calm and focused user experience.

### ⚡ Lightweight Performance
Since the project is built using basic web technologies, it loads quickly and performs smoothly without requiring heavy frameworks or libraries.

---

## 🛠️ Technologies Used

This project is built using fundamental web development tools:

- **HTML5** – for structure and content layout
- **CSS3** – for styling, layout, and responsiveness
- **JavaScript (optional)** – for interactivity and dynamic behavior
- **GitHub Pages** – for deployment and hosting

These technologies were chosen intentionally to keep the project lightweight, beginner-friendly, and easy to maintain.
